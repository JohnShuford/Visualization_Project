About this project


The data used in this project is from a dataset of trending YouTube videos in the years 2017-2018. This project examines the trending YouTube videos in 10 different countries: United States, Russia, Canada, Great Britain, France, Germany, Japan, South Korea, India, and Mexico. This interactive website allows users to explore the dataset with dashboards, interactive visualizations, charts, and maps. This website also incorporates machine learning, where the user can enter a hypothetical video title that will run through our classifier model, returning the predicted category that the video would fall into based on the title content.
